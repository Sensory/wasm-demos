# Sensory WASM Demos

Live browser-based demos of [Sensory's](https://www.sensory.com) voice and speech AI technology, powered by WebAssembly (WASM). No installation required — run directly in your browser.

Sensory's technology runs entirely **on-device** with no cloud dependency, ensuring low latency, privacy, and offline capability.

---

## Demos

### 🎙️ Wake Word Detection
Detect a custom wake word in real time using Sensory's on-device speech recognition engine.

👉 [Try the Wake Word Demo](https://voicehub.sensory.com/demo-ww)

---

### 📝 Speech to Text
Convert spoken audio to text in real time, entirely on-device.

👉 [Try the Speech to Text Demo](https://voicehub.sensory.com/demo-stt)

---

### 🔍 Command & Search
Recognize spoken commands and search terms using Sensory's embedded voice AI.

👉 [Try the Command & Search Demo](https://voicehub.sensory.com/demo-cs)
